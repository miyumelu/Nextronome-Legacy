# Nextronome
An accompaniment system with the intention of creating a Korg-like accompaniment.

This is the legacy version, which is based on .NET Framework 4.7.2.

This has been replaced by [Nextronome 2](https://github.com/miyumelu/nextronome-2).

The legacy version will continue to be developed until it has full functionality. After that, everything will run on [Nextronome 2](https://github.com/miyumelu/nextronome-2), and only bug fixes will be released.
